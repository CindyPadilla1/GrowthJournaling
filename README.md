# Growth Journaling

This journal is a private, aesthetic, emotional journaling app that helps users reflect gently on emotional patterns over time.

## Features
- Emotion-based journaling
- Sentiment analysis
- Animated growth system
- Weekly emotional summaries
- Dark mode & mood-based gradients
- Local-only privacy

## Tech Stack
- HTML, CSS, JavaScript
- LocalStorage

## Privacy
All data stays on your device
